## Projects Detail

### Expanding Cards
**Interactive cards that expand smoothly when clicked** — A beautiful UI component showcasing famous places in Saint Petersburg, Russia.

| Feature | Description |
|---------|-------------|
| Click Interaction | Click any card → it expands, others shrink |
| Smooth Animation | CSS transitions with `cubic-bezier` easing |
| Fully Responsive | Desktop, tablet, and mobile layouts |
| Visual Effects | Gradient text, shadow effects, fade-in animation |
| 7 Landmarks | Saint Isaac's Cathedral, Peterhof Palace, Hermitage Museum, and more |

**Tech Stack**: HTML5, CSS3, Vanilla JavaScript

**Live Demo**: [View Demo](https://nguyenduongitmo.github.io/petprojects/expanding-cards/)

## Responsive Design

All projects follow **Mobile First** approach — styles are written for mobile by default, then enhanced for larger screens.

| Breakpoint | Target | Media Query |
|------------|--------|-------------|
| < 768px | Mobile (default) | No media query needed |
| ≥ 768px | Tablet | `@media (min-width: 768px)` |
| ≥ 1024px | Desktop | `@media (min-width: 1024px)` |
| ≥ 1440px | Large Desktop | `@media (min-width: 1440px)` |
