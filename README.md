## Responsive Design

All projects follow **Mobile First** approach — styles are written for mobile by default, then enhanced for larger screens.

| Breakpoint | Target | Media Query |
|------------|--------|-------------|
| < 768px | Mobile (default) | No media query needed |
| ≥ 768px | Tablet | `@media (min-width: 768px)` |
| ≥ 1024px | Desktop | `@media (min-width: 1024px)` |
| ≥ 1440px | Large Desktop | `@media (min-width: 1440px)` |

## What I've Learned
### Expanding Cards
- CSS `flex` property for dynamic card sizing
- CSS transitions with `cubic-bezier` for smooth animations
- `background-size: cover` and `background-position` for image handling

- Responsive breakpoints for multiple devices
- Keyframe animations (`fadeInDown`)
